# NearBuy 

Running instuctions - back-end: 

  Install Python3.4:
    https://www.python.org/downloads/
    
  Check you have correct version:
    python --version

  Launch virtualenv:
    cd back-end
    source py3env/bin/activate
 
  Run app.py:
    python app.py

  To deactivate env:
    deactivate
